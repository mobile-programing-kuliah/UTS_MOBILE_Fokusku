# Fokusku - Aplikasi Manajemen Fokus (Pomodoro)

**Fokusku** adalah sebuah prototipe aplikasi Android yang dirancang untuk membantu pengguna meningkatkan produktivitas melalui teknik manajemen waktu (seperti Pomodoro). Aplikasi ini menyediakan antarmuka yang bersih dan menenangkan dengan tema *Earth Tone* untuk mendukung konsentrasi saat belajar atau bekerja.

## Deskripsi Aplikasi
Fokusku memungkinkan pengguna untuk merencanakan sesi fokus mereka dengan memilih durasi per sesi dan target jumlah sesi yang ingin dicapai. Aplikasi ini secara otomatis menghitung waktu istirahat (istirahat pendek dan istirahat panjang) sesuai dengan rencana yang dipilih.

## Desain Antarmuka
Aplikasi ini menggunakan bahasa desain **Material 3** dengan sentuhan kustom:
- **Warna Utama**: Earth Tone (Hijau Hutan, Hijau Lembut, dan Krem).
- **Tipografi**: Menggunakan font sans-serif yang bersih dan tebal.
- **Aksesibilitas**: Desain responsif yang menyesuaikan berbagai ukuran layar Android.
